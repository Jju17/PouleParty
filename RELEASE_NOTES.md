# Release 1.2.1

## What's New

### Improved Game Creation
- Final zone glow always visible on the chicken's map during gameplay
- Faster map loading in game creation (lighter map style + pre-warming)
- Better recap view: shows role, max players, end time, power-up types, total price
- Tapping outside the zone in final-zone mode now re-sets the start zone
- Zone config description bar with contextual instructions
- Fixed header and bottom bar layout — content no longer overlaps navigation

### Bug Fixes
- Fixed progress bar showing an orange dot at the end (Android)
- Fixed zone configuration not being detected after setting zones (iOS)
- Fixed map pins (START/FINAL) disappearing on zoom (iOS)
- Fixed Firebase AppDelegate swizzler warning (iOS)
